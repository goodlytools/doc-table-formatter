# doc-table-formatter

Public site and policies for the **Doc Table Formatter** Google Docs add-on.

Doc Table Formatter is a lightweight Google Docs add-on that styles tables in Google Docs from a sidebar. It runs entirely inside the document you currently have open; no data is stored or transmitted to external servers.

## Features

- Bold text styling for Header Row, Total Row, First Column, and Last Column
- Header fill color with a preset palette plus a full custom color picker (hex supported)
- Row banding or Column banding with customizable odd/even colors and a one-click swap button
- Column Width tools — Distribute Columns evenly, or Best Fit to minimize overall table height
- One-click Apply and Clear formatting

## Usage

1. Install Doc Table Formatter from the Google Workspace Marketplace.
2. Open any Google Doc.
3. Click **Extensions → Doc Table Formatter → Open sidebar**.
4. Place your cursor in a table and use the sidebar controls to style it.
