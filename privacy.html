Privacy Policy

Last updated: May 7, 2026

Doc Table Formatter ("the Add-on") is a Google Docs editor add-on that helps users format tables inside their own Google Docs. This Privacy Policy explains what data the Add-on accesses and how it is handled.

1. Information We Access

The Add-on uses the OAuth scope https://www.googleapis.com/auth/documents.currentonly, which only allows access to the single Google Doc the user has explicitly opened the Add-on in. The Add-on reads and modifies table formatting (colors, banding, bolding, padding) in that document only when the user clicks a button in the Add-on sidebar.

2. Information We Do Not Collect

The Add-on does not collect, store, transmit, sell, or share any personal data, document content, or usage analytics. No data leaves the user's Google account. There is no server component; all logic runs inside Google Apps Script under the user's own Google session.

3. Data Storage

The Add-on does not persist any user data. It does not use Google Apps Script's PropertiesService, CacheService, or any external database to store user content.

4. Third Parties

The Add-on does not share data with third parties. It does not use external APIs, analytics, advertising networks, or tracking services.

5. Children's Privacy

The Add-on is not directed to children under 13 and does not knowingly collect information from them.

6. Changes to This Policy

We may update this policy from time to time. The "Last updated" date above will reflect the most recent change.
